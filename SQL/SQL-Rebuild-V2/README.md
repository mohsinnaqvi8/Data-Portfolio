# SQL Rebuild V2

## Overview

SQL Rebuild V2 is a comprehensive SQL portfolio project built using the **Instacart Market Basket Analysis** dataset. The project follows a structured progression from SQL fundamentals to advanced analytical techniques commonly used in data analytics and data engineering.

The notebook focuses on solving real-world business problems through clean, modular, and production-style SQL queries while emphasizing analytical thinking, query optimization, and reusable SQL patterns.

---

## Objectives

- Strengthen analytical SQL problem-solving skills.
- Practice writing clean, modular, and maintainable SQL.
- Apply SQL to realistic business and reporting scenarios.
- Master intermediate and advanced SQL concepts.
- Build production-style analytical workflows using real-world data.

---

## Dataset

**Dataset:** Instacart Market Basket Analysis

The project analyzes anonymized online grocery shopping transactions, including:

- Customers and orders
- Products
- Aisles and departments
- Reorder behavior
- Order history

The dataset provides a realistic environment for solving analytical business problems involving customer behavior, product performance, ranking, aggregation, and reporting.

## SQL Concepts Covered

### Core SQL
- Filtering (`WHERE`)
- Sorting (`ORDER BY`)
- Aggregate Functions
- `GROUP BY`
- `HAVING`
- Aliasing

### Intermediate SQL
- Joins
- `CASE WHEN`
- Common Table Expressions (CTEs)
- Scalar Subqueries
- Nested Subqueries

### Advanced SQL
- Window Functions
- `ROW_NUMBER()`
- `RANK()`
- `DENSE_RANK()`
- `LAG()`
- `LEAD()`
- Self Joins
- Running Totals

### Analytical Applications
- Product & Department Analysis
- Basket Analysis
- Percentage Contribution Analysis
- Above/Below Average Analysis
- KPI Reporting
- End-to-End Analytical Queries
